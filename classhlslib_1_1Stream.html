Detailed Description

template<typename T, unsigned capacityDefault, Storage storage>
class hlslib::Stream< T, capacityDefault, storage >

Custom stream implementation, implementing thread-safe and blocking Push and Pop, as well as read/write conforming to the hls::stream interface. The capacityDefault template parameter is a convenience override used when instantiating arrays of streams, as the capacity cannot be passed to the constructor then. The constructor argument is always favored over the template argument, if specified.
