13 // This header provides functionality to simulate dataflow functions that
14 // include loops in conjunction with Stream.h.
15 //
16 // Dataflow functions must be wrapped in the HLSLIB_DATAFLOW_FUNCTION macro,
17 // which will launch them as a C++ thread when running simulation, but will
18 // fall back on normal function calls when running synthesis.
19 //
20 // The macro HLSLIB_DATAFLOW_FINALIZE must be called before returning from the
21 // top level function to join the dataflow threads.
22 // HLSLIB_DATAFLOW_INIT currently has no purpose, but is included for symmetry.
23 //
24 // TODO: HLSLIB_DATAFLOW_FUNCTION does not work when calling templated functions
25 // with multiple arguments, as it considers the comma a separator between
26 // function arguments. Look into alternative implementation, or always use
27 // variadic templates if newer Vivado HLS start supporting inlining the
28 // called function (it currently fails with "no function body" on the
29 // dataflow function in question).
