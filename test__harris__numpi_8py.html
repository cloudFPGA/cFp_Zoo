Detailed Description

A python script for testing the cF harris kernel. The script takes as argument the fpga ip, the port and the numpi array of an image to be processed. This array should be an 1-D array, containing all pixels of a CV MAT in CV_8UC1. The kernel will rerurn a numpi array whoch is the array with only the detected points.
